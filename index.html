Stillness

Stillness is a minimal and clean Wordpress theme based on z2h.

View on GitHub Download .zip Download .tar.gz

Close your eyes

Years ago @leobabauta released the Zen Habits theme which I really liked for its minimal approach and style. In 2013, he released v2.0 and now, three years later, I decided to rework it. This reworking was in two parts: design and functionality. Specifically, the original z2h theme required the user to dive into the code to make certain changes to get the theme to work as expected. Stillness does away with that by ensuring that everything can be done from the Wordpress admin menu following some simple set-up instructions.

Stay still

This theme stays true to the original version, being minimal, and comes with a great deal of stylistic changes. Functional changes include moving archival links to the footer1, h1 tagging post titles for better SEO, improved CSS classes, a better HTML structure, support for Wordpress Menus and, in what is perhaps the biggest fork off the spirit of the original theme, there is exactly one option that lets users pick an accent colour of their choice to be used for links.

Also, the archival link on the original theme has been changed to regular Wordpress pagination because it was one of the features that required the user to change bits of code. I'm hoping to bring back the feature in a later version of this theme if I can think of a way to make it work using just Wordpress2. Update: Done. The archival link now works almost exactly as on z2h with one difference: in order to keep things minimal, it cascades all previous posts without monthly separation.

Lastly, pages and posts have different archival links and, rather intelligently, the archival link is not displayed on the archival page. Also, single post pages (neither archives nor the front page) will carry the author's byline.

Breathe deep

I may update this theme in the future. Since it uses Google Fonts, you should have no support to get the theme looking exactly as you see on the demo site. No support will be provided. However, installation instructions are available and I have set up a demo site so you can see exactly how things will work. I usually reply to all e-mails, so if you really have to, do get in touch with me.
